HTML es un lenguaje de marcado que se utiliza para estructurar y dar formato a los documentos web. Es el esqueleto de una página web y define su estructura y contenido.

Etiquetas HTML

Las etiquetas HTML son las palabras clave entre ángulos (<>) que se utilizan para crear elementos HTML. Hay una gran variedad de etiquetas HTML que puedes usar para crear diferentes estructuras y estilos en tus páginas web.

Ejemplos de etiquetas HTML

  • Encabezados: <h1> hasta <h6>
  • Párrafos: <p>
  • Enlaces: <a>
  • Imágenes: <img>
  • Listas: <ul>, <ol> y <li>
